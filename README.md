# World Map of Castles

A map with the location of castles around the world for Python 3.8

Website where data was extracted: http://www.topcastles.com/

---

## Modules/Libraries
* Folium
* pandas
* bs4 (BeautifulSoup)
* requests
* re
* lxml (etree)
* random
* json
* time (sleep)
* os
